Software

 

The toolkit was developed using R: A Language and Environment for Statistical Computing and shiny: Web Application Framework for R. Additional R packages used to support the toolkit include:

  • data.table: extension of 'data.frame'
  • dplyr: a grammar of data manipulation
  • DT: a wrapper of the JavaScript library 'data.tables'
  • RColorBrewer: ColorBrewer palettes
  • readxl: read excel files
  • ggplot2: create elegant data visualisations using the grammar of graphics
  • grid: the grid graphics package
  • gridExtra: miscellaneous functions for "grid" graphics
  • tidyr: easily tidy data with 'spread()' and 'gather()' functions

The standalone version of the toolkit uses a portable version of R and the portable edition of the Mozilla Firefox web browser.
